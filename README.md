# Mahjong_Scorebook Web App
## Overview: 
A web-based score management system designed for the traditional game Mahjong, which has a unique and complex scoring method.
This app allows the players to record, calculate and store their scores across multiple rounds and sessions, ensuring accuracy and ease of access. 

## Key Features:
- **Automated Score Calculation**: Implement Mahjong's unique scoring rules
- **Player Management**: Track players across mumole sessions
- **Session History**: View all past game results
- **Leaderboard**: Ranks players based on their total scores
- **Consistent Data**: All data are saved and retrieved from MySQL

## Tech Stack:
- **Frontend**: HTML, CSS/SCSS, JavaScript
- **Backend**: Python Flask (Web Framework)
- **Database**: MySQL
- **Version Control**: Git
